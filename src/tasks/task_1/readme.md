# Задача для 1 семестра по курсу "системная инженерия"

Необходимо разделиться на группы по 3 человека. Индивидуальные проекты не допускаются. Проекты по 2 человека получают оценку ниже

Выбирается тема - одна из предложенных ниже. По теме необходимо выполнить набор задач

Суть задачи: необходимо спроектировать систему, описав как она будет работать. К вопросу необходимо подойти комплексно - от формирования требований к проработке архитектуры и до описания тестовых сценариев

Что наиболее важно отразить в работе:
- Суть задачи, требования
- Рассмотреть ее с точки зрения разного набора стейкхолдеров 
- Рассмотреть все интеграции, обеспечивающие системы 
- Подробно сформулировать NFR
